# Security Policy

## Reporting a Vulnerability

<!-- TODO: Update with Melodarr-specific contact details when available -->

Please report (suspected) security vulnerabilities on Discord (preferred) to
any of the Melodarr Dev role holders or via email: development@servarr.com. You will receive a response from
us within 72 hours. If the issue is confirmed, we will release a patch as soon
as possible depending on complexity/severity.
